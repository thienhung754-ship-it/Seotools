# FinSEO Pro - Tạp chí Kinh tế - Tài chính

Công cụ tối ưu nội dung chuẩn SEO chuyên dụng cho mảng Tài chính và Báo chí.

## 🚀 Tính năng nổi bật
- **AI-Native Editor:** Soạn thảo chuyên nghiệp với thanh công cụ WYSIWYG.
- **Scoring System:** Chấm điểm SEO khắt khe theo tiêu chuẩn ban biên tập.
- **Multi-Format:** Hỗ trợ nạp file PDF (Text-based), DOCX, MD, HTML và TXT.
- **Auto-Fix:** AI tự động sửa bài theo các gợi ý cải thiện.
- **Smart Meta:** Bộ đếm ký tự Meta Description chuẩn SEO (150-160 ký tự).

## 🛠️ Cấu hình & Sử dụng
1. Mở file `app.js`.
2. Tìm biến `apiKey` trong `state` để thay đổi hoặc sử dụng key đã có sẵn.
3. Chạy trực tiếp `index.html` trên trình duyệt hoặc qua Live Server.

## 📂 Cấu trúc dự án
- `index.html`: Giao diện người dùng.
- `style.css`: Hệ thống Style Apple Minimalist.
- `app.js`: Logic xử lý AI và bóc tách dữ liệu.
